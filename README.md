## **Moon Shop**
----
> **프로젝트** : 쇼핑몰 사이트
> **프로젝트 개요** : 개인 공부한 내용을 직접 사용하고 관심있는 분야인 이커머스 시스템을 직접 구현하기 위해 시작
기획 및 개발: 문 승 주
제작 기간: 2023.12.28 ~
주요 기능: 상품 관리, 주문 관리, 쿠폰 기능, 리뷰 기능 등 쇼핑몰의 기본적인 서비스를 구현
사용 기술
- front: JavaScript, Thymeleaf
- back: Spring Boot, JPA, Spring Security, queryDSL
