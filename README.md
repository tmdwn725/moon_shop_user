# **Moon Shop**
----
>### **프로젝트 개요**
>개인 공부한 내용을 직접 사용해보고 관심있는 분야인 이커머스 시스템을 직접 구현하기 위해 시작하였습니다.
>
>### **기획 및 개발**
>문승주
>
>### **제작 기간**
>2023.12.28 ~
>
>### **주요 기능**
