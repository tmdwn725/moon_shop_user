# **Moon Shop**
----
>### **프로젝트 개요**
>개인 공부한 내용을 적용하여 관심있는 분야인 이커머스 시스템을 직접 구현하기 위해 시작하였습니다.
>
>### **기획 및 개발**
>문승주
>
>### **제작 기간**
>2023.12.28 ~ 2024.01.30
>
>### **주요 기능**
>상품 관리, 주문 관리, 쿠폰 기능, 리뷰 기능 등 쇼핑몰 서비스 구현
>
